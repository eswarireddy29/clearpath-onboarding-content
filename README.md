# ClearPath — Onboarding Content Design

## Overview
Content redesign for a B2B SaaS analytics dashboard 
serving 3 distinct user types — Admin, Analyst, and Field Agent.

## Problem
The existing dashboard used technical jargon that confused 
non-technical users, leading to high support ticket volume 
during onboarding.

## My Role
UX Writer & Content Designer — conducted content audit, 
mapped copy to personas, rewrote all onboarding strings, 
delivered annotated Figma handoff.

## What's Inside
- Content audit of 40+ UI strings
- Persona mapping (Admin, Analyst, Field Agent)
- Full copy rewrite with tone rationale
- Annotated Figma handoff document

## Key Results
- 35% reduction in time-to-first-insight (usability testing)
- Zero developer clarification requests after handoff

## Tools Used
Figma, Notion, Google Sheets
